What is the website designed for?
This is a project for visualizing different aspects of the Trento tree dataset. The website aims to provide readable and appealing visualizations for the reader.


What is the Trento tree dataset?
This dataset provides some very basic yet important details on the trees in Trento, Italy. We can learn a lot from this data, including the species, height, circumference, crown breadth, and geographic coordinates, to name just a few.


What is the final goal of this project?
Make the necessary calculations, and then link the "price" to several (ecological) advantages that trees provide.
